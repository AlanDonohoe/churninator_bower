# The Churninator

This is The Churninator - The Killer of Churn.

## Installation

Add the dist/main.js file to your javascript files in your own way...


## Usage
On your cancellation page, give you cancel button an id of 'cancel'

In your JS, run:
```javascript
churninator.churninate();
```
